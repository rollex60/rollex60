# Netwods — Criação de Sites Profissionais em Portugal

Sou Yevhen Drahan, fundador da **Netwods**, e ajudo empresas, pequenos negócios e projetos locais a criar uma presença online clara, moderna e preparada para gerar contactos.

Na Netwods desenvolvo websites institucionais, landing pages, lojas online e soluções digitais focadas em design, velocidade, SEO básico e facilidade de utilização.

## O que faço

- Criação de websites profissionais
- Landing pages para campanhas e pedidos de orçamento
- Lojas online com WooCommerce ou Shopify
- Otimização básica para Google
- Estruturação de páginas com foco em conversão
- Manutenção e melhorias em sites WordPress

## Tecnologias e ferramentas

**Web**
HTML · CSS · JavaScript · React · Bootstrap

**WordPress**
WordPress · WooCommerce · Elementor · Flatsome · SEO técnico

**Back-end**
Python · Django · APIs · Bases de dados

**Marketing e SEO**
Google Search Console · Google Analytics · Otimização On-page · Performance
